## 🇧🇷 Português

Filmeflix é uma interface de streaming altamente refinada feita com as melhores práticas de Frontend. O projeto apresenta um design premium, transições de páginas suaves, carrosséis responsivos, efeito dinâmico *"Ambilight"* (onde a cor do pôster reflete no fundo) e um sistema de memória local para favoritos e filmes recentes, tudo isso sem a necessidade de um banco de dados ("backend").

### Funcionalidades ✨
- **Premium Dark Mode UI**: Inspirado pelos melhores serviços de streaming, usando tons de preto e dourado.
- **Dynamic Ambilight**: O plano de fundo reage sutilmente à paleta de cores dominante da página do filme.
- **Estado via LocalStorage**: Carrosséis de "Minha Lista" e "Continuar Assistindo" 100% funcionais usando a memória do navegador.
- **Framer Motion**: Transições e animações fluidas ao navegar pelas rotas do aplicativo.

### Tecnologias 🛠️
Construído com React 19, TypeScript, Vite, Tailwind CSS v4 e React Router DOM.

#### Como rodar localmente:
```bash
npm install
npm run dev
```

## 🇺🇸 English

Filmeflix is a highly refined streaming interface mockup built with Frontend best practices. The project features a premium design, smooth page transitions, responsive carousels, dynamic "Ambilight" CSS effects (where the poster's accent color bleeds into the background), and a local memory system for favorites and recent movies—all without relying on a back-end database.

### Features ✨
- **Premium Dark Mode UI**: Inspired by top-tier streaming platforms, using deep black and gold accent tones.
- **Dynamic Ambilight**: The background subtly reacts to each movie's dominant color palette.
- **LocalStorage State**: Fully functional "My List" and "Keep Watching" components using browser memory.
- **Framer Motion**: Fluid entry/exit and sliding animations between application routes.

### Technologies 🛠️
Built with React 19, TypeScript, Vite, Tailwind CSS v4, and React Router DOM.

#### How to run locally:
```bash
npm install
npm run dev
```
